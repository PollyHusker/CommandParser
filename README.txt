CommanParser/bin/Release/CommandParser.exe -консольная утилита.
Для выхода из программы использовать стандартную команду ctrl+c.
С учетом отсутствия тесной взаимосвязи между различными параметрами
 предпроверка правильности параметров минимальная: проверяется только отсутствие некорректных ключей.
О неправильном наборе аргументов для ключа пользователь уведомится уже в момент выполнения команды. 
Таким образом для пользователя будет очевиднее для какого именно ключа были заданы некорректные аргументы 
(так как ключи могут повторяться) - отработают все ключи вплоть до ключа, для которого заданы некорректные аргументы.
Символ кавычек " " является спец.символом и используется для задания в качестве аргумента ключа нескольких слов.
Например для печати hello there! -print "hello there!".
-k "half life 3" "hl 3" выведет:
half life 3 - hl 3
